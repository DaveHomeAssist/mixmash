What's in the Set

Built for the Festival Floor

Platform fighting meets festival culture. Every mechanic is tuned like a headliner set — fast, punchy, and impossible to walk away from.

12 DJ Fighters
A 12-fighter roster led by deadmau5, Skrillex, Marshmello, Daft Punk, Tiësto, Subtronics, Avicii, Excision, Zedd, Aphex Twin, Calvin Harris, and Rezz. Each fighter has a distinct weight, movement profile, and grab-heavy matchup rhythm.
9 Stage Lineup
The current build spans Battlefield, Final Destination, Skybridge, Festival Stage, Tomorrowland, Ultra, Burning Man, Coachella, and EDC. Some are flat duel spaces, others lean into moving platforms and festival spectacle.
Deep Combat
DI, grabs, spot dodges, aerial dodges, clang system, hitstop, and knockback scaling. Frame-tight mechanics with the depth to go competitive. This isn't a party game — unless you want it to be.
Gamepad + Keyboard
Full gamepad API support with rumble feedback that scales to damage. Keyboard controls for both players. Plug in, pick up, throw down.
3 Game Modes
Stock Match (last one standing), Time Match (highest KO count wins), and Training Mode with hitbox visualization, frame data, and damage resets. Warm up or go all in.
CPU AI (3 Levels)
Level 1 for casual vibes, Level 3 for a real challenge, Level 5 for when you want to sweat. The AI reads your habits, shields, grabs, and DIs like a seasoned bracket player.
